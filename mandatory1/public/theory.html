Node.js

Node.js is a web server framework using JavaScript. It allows dynamic content, CRUD operations on files and data and/or collecting data from forms. Its advantage is that it is asynchronous and runs concurrently: it uses a single thread in the app, that gives work to background multiple threads.

In Node.js the task is sent from client to node.js server. The Node.js server lets the file system deal with processing the file. Meanwhile Node.js server is ready to take another request. Finally, after the file was read by the file system it is returned by Node.js from file system to client.
